# 一个交互性的网易云音乐爬虫

## 功能介绍

#### 下载音乐

- 下载单曲
- 按歌单下载歌曲
- 按歌手下载音乐

#### 下载评论

- 单曲热评/全评
- 歌单热评/全评

#### 歌手粉丝地域分布
- 调用百度地图api实现
-　ak字段需要自己到(百度地图)[http://lbsyun.baidu.com/]申请
